Treasure Web Hunt is a simple game that brings visits to your website. A random treasure will appear on your website an people came on your website to collect it.

== Description ==

Treasure Web Hunt is a simple and funny game. It's a World Wide Treasure Hunting Game that made people browse a lot of websites searching for a treasure. The main goal of this funny little game is to collect the biggest number of treasures.

Playing is very simple:

* Get your account or use your facebook account at www.treasurewebhunt.com
* Browse randomly displayed treasure websites or find a treasure on the web by your own 

Adding a treasure to the list is very simple too:

* Install this plugin
* Go to the option page to get and insert a free Treasurewebhunt.com Code

All done! 
Enjoy yourself!

== Installation ==

1. Upload `treasure-web-hunt` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the admin panel to get a free Treasurewebhunt.com Code to activate your treasure.
4. Let people visit your site to collect the treasure!
